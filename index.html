Research Engineer Intern

As a Research Engineer Intern, my job was to gather data and implement various machine learning algorithms which were implemented in our products. I developed different classification and regression algorithms using frameworks such as Tensorflow and Sci-kit Learn which helped the company determine the state of health of Li-ion batteries. This involved lots of data analysis and I sped up the process by developing software tools using Python to automate the process.
