Cadex Electronics Inc.

Research Engineer Intern

As a Research Engineer Intern, my job was to gather data and implement various machine learning algorithms which were implemented in our products. I developed different classification and regression algorithms using frameworks such as Tensorflow and Sci-kit Learn which helped the company determine the state of health of Li-ion batteries. This involved lots of data analysis and I sped up the process by developing software tools using Python to automate the process. I automated the data gathering process on up to 20 machines simultaneously by using multithreading and a publish/subscribe framework and developed a GUI that other Engineers could use to speed up the process of gathering data. The data was stored in a database which I used Django to interface with and ran the webserver with Apache using a Linux VM. I also developed prototype firmware for DSP applications using an ARM Cortex M7 using STM32CubeMX and IAR as well as for TI products using Code Composer Studio.
